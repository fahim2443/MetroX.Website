About MetroX

Built for Dhaka Metro Rail and Rapid Pass users, MetroX keeps everyday commuting effortless. Readers can quickly scan their NFC cards, browse every MRT line and station, and calculate fares before tapping in.

  • Full MRT network overview with station insights
  • Route and fare calculator with travel time estimates
  • Dark/light theme, multi-language support, and offline viewing

Supported Cards

Dhaka MRT Rapid Pass, Metro Rail Travel Cards, and any FeliCa-based transit cards used in Bangladesh.

Tech Stack

React Native + Expo · NFC (FeliCa) · Interactive SVG Map · Material Design UI
